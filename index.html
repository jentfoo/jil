JIL

JIL is a java Image Library made to help make image operations in java simpler. It is currently still in alpha and should be used as such. JIL is Licensed under the LGPL 3.0 licesnse.

Compiling JIL

JIL uses the gradle build system and compiling and running its unit tests should be very easy.

To Build run:

$ cd jil
$ gradle build

This will build the software, javadocs and run the unit tests. All compiled files will be put into the build directory. The jar file is located in build/libs and the javadocs are in build/docs/javadoc.

If you prefer to just build the jar you can run:

$ cd jil
$ gradle assemble

And of course to clean run

$ cd jil
$ gradle clean
